# 🍅 ポモドーロタイマー

シンプルで使いやすいWebベースのポモドーロタイマーアプリケーションです。

## 機能

- **カスタマイズ可能な設定**
  - 作業時間（1〜60分）
  - 休憩時間（1〜30分）
  - 長い休憩時間（1〜60分）
  - セット数（1〜12セット）
  - 長い休憩の頻度

- **タイマー機能**
  - 自動セッション進行（作業 → 休憩 → 作業...）
  - 一時停止/再開
  - リセット
  - バックグラウンド動作対応

- **通知機能**
  - 音声通知（作業終了/休憩終了/全セット完了）
  - ブラウザ通知（デスクトップ通知）
  - 音量調整・ミュート機能

- **データ永続化**
  - 設定をLocalStorageに保存
  - ブラウザを閉じても設定を保持

## 技術スタック

- **言語**: TypeScript
- **ビルドツール**: Vite
- **パッケージマネージャー**: Bun
- **フレームワーク**: Vanilla JS（フレームワークレス）

## セットアップ

### 前提条件

- [Bun](https://bun.sh/) がインストールされていること

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd pomodoro-timer

# 依存関係をインストール
bun install
```

### 開発サーバーの起動

```bash
bun run dev
```

ブラウザで http://localhost:5173 を開いてください。

### ビルド

```bash
bun run build
```

ビルド成果物は `dist/` ディレクトリに出力されます。

### プレビュー

```bash
bun run preview
```

## 使い方

1. **設定画面**で作業時間、休憩時間、セット数などを設定
2. **スタート**ボタンをクリックしてタイマーを開始
3. タイマーが自動的にセッションを進行
4. 各セッション終了時に音声とブラウザ通知でお知らせ
5. 全セット完了で達成メッセージを表示
